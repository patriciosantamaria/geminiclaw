# Common Responses & Phrasings

## Technical Updates (To Management)
- **Status (Looker Studio):** "Just a quick update—our current Looker Studio report is outdated since data ingestion paused late last year."
- **New Queries:** "To get accurate numbers, we'd need to set up a new query from the latest Workspace logs. Let me know if this is a priority to pick up now."
- **Data Validation:** "Since these metrics are strategic, we might want to ask a Data Analyst to help validate the new queries."

## Declining Requests (The "Patricio NO")
- **Individual Requests:** "I'd love to help out, but setting this up as a one-off might delay our other deliverables. Could we consider..."
- **Alternative Path:** "Perhaps [Name] could sync with the data analytics team to build the SQL queries they need for this report?"
